# Build Guide: "AI Power Tools" Extension

In this workshop, we will build a production-ready Chrome Extension.

This guide is broken down into 4 parts. Follow them in order to build your extension.

## 1. [Manifest & Configuration](guide-manifest.md)
Set up the project structure and permissions.

## 2. [Background Script](guide-background.md)
Implement the Context Menus ("Right-click to Translate").

## 3. [Rewrite Button](guide-content-script.md)
Create the smart AI button that injects into web pages.

## 4. [Side Panel AI](guide-sidepanel.md)
Build the persistent Chat Assistant and Tools interface.
