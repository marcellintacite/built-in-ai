Getting Started

  1. Install the Andika AI extension from the Chrome Web Store
  2. Ensure you have Chrome version 127 or higher
  3. Enable Chrome's Built-in AI in chrome://flags
  4. Navigate to any webpage with text inputs
  5. Focus a text field and click the golden AI button that appears
Note: Andika AI requires Chrome's built-in Gemini Nano model. Make sure "Prompt API for Gemini Nano" is enabled in chrome://flags.

Frequently Asked Questions

Q: Is my data private?
Yes! All AI processing happens locally on your device using Chrome's built-in Gemini Nano. Your text never leaves your computer. No data is sent to external servers, and we don't collect any user information.
Q: Why isn't the AI button appearing?
Make sure you've enabled Chrome's Built-in AI in chrome://flags. Search for "Prompt API for Gemini Nano" and set it to "Enabled". Then restart Chrome and reload the extension.
Q: Does this work on all websites?
Yes! Andika AI works on any website with text input fields, including Gmail, LinkedIn, Twitter, forms, and more. The AI button appears automatically when you focus a text field.
Q: Is Andika AI free?
Yes, Andika AI is completely free with no subscriptions, no hidden costs, and no premium tiers. All features are available to everyone.
Q: What can I do with the sidepanel?
The sidepanel provides an advanced chat interface where you can have extended conversations with AI, send images for analysis, and customize your writing preferences (tone, format, length).
Q: How do I change the AI's writing style?
Click the extension icon to open the sidepanel, then go to the Settings tab. You can customize the tone (casual/formal), format (plain text/markdown), and length (short/medium/long).

Troubleshooting

Extension Not Working

  1. Check that you're using Chrome 127 or higher
  2. Enable "Prompt API for Gemini Nano" in chrome://flags
  3. Restart Chrome completely
  4. Reload the extension from chrome://extensions

AI Button Not Appearing

  1. Make sure the extension is enabled in chrome://extensions
  2. Try refreshing the webpage
  3. Check browser console for any errors (F12)
  4. Verify you're not on a restricted page (chrome://, chrome-extension://)

Slow Performance

All AI processing happens on your device. Performance depends on your computer's capabilities. For best results, ensure you have sufficient RAM and CPU resources available.
